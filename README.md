# To-Do List Web Application

This project is a simple and interactive To-Do List web application that allows users to add, complete, and remove tasks dynamically. The application is built using HTML, CSS, and JavaScript, focusing on DOM manipulation and event-driven programming to manage user interactions efficiently.

The application provides a clean interface where tasks can be added to a list, marked as completed, or removed when no longer needed.

# Features

Add new tasks to the list

Mark tasks as completed by clicking on them

Delete tasks from the list

Input validation to prevent empty tasks

Dynamic task management using JavaScript DOM manipulation

Tech Stack

HTML – Structure of the application

CSS – Styling and layout

JavaScript – Application logic and DOM manipulation

# How It Works

The user enters a task in the input field.

When the Add Task button is clicked:

The task is added to the list dynamically.

Clicking on a task toggles its completed status.

Each task has a delete button that removes it from the list.
